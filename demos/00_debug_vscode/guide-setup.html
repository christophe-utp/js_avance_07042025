Prérequis

  • Visual Studio Code installé
  • Extension Live Server (par Ritwick Dey)
  • Extension Debugger for Chrome (ou équivalent intégré)

Lancer le projet

  1. Ouvre le dossier du projet dans VS Code
  2. Clic droit sur index.html → "Open with Live Server"
  3. Vérifie l’URL ouverte (souvent http://127.0.0.1:5500)

Configurer le débogueur

Crée un fichier .vscode/launch.json avec le contenu suivant :

{
  "version": "0.2.0",
  "configurations": [
    {
      "type": "pwa-chrome",
      "request": "launch",
      "name": "Déboguer dans Chrome",
      "url": "http://localhost:5500",
      "webRoot": "${workspaceFolder}"
    }
  ]
}
