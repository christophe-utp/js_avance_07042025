Breakpoints

Posez un point d'arrêt sur une ligne de code pour arrêter l'exécution à cet endroit.

Step Over / Step Into / Step Out

Pas à pas : suivez l'exécution ligne par ligne ou entrez/sortez d'une fonction.

Watch

Ajoutez une variable à surveiller dans le panneau Watch pour suivre sa valeur.

Call Stack

Visualisez la pile d'appels actuelle pour comprendre l'enchaînement des fonctions.

Breakpoints conditionnels

Arrêtez l'exécution uniquement si une condition spécifique est remplie.

Modifier une variable en live

Dans le panneau Variables, cliquez sur une valeur pour la modifier en temps réel.

Asynchrone

Déboguez du code avec setTimeout, Promises ou async/await grâce à la file d'attente d'évènements.
