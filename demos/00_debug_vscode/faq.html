Pourquoi mon breakpoint ne fonctionne pas ?

Assurez-vous que le code est bien chargé dans la page et que la source correspondante est visible dans les DevTools ou VS Code.

Je ne vois pas mes variables dans Watch

Vérifiez que vous êtes à un moment de l'exécution où les variables sont bien accessibles (dans le bon scope).

Step Into ne rentre pas dans la fonction

Si la fonction est native (comme console.log), VS Code ne peut pas y entrer. Essayez sur une fonction utilisateur.

Comment lancer un débogage web dans VS Code ?

Utilisez une configuration de type "pwa-chrome" et ouvrez la bonne URL locale.
